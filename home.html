MORSE_ALPHABET

International Morse Code. The foundation of modern telecommunication. Dots, dashes, and the rhythm of the pulse.

📡

HOW_TO_USE

Press the telegraph key (or Space bar) briefly for a dot, hold longer than 250 ms for a dash. Pause 0.6 s to complete a letter, 1.4 s to add a word space. The translation updates in real time.

⚡

AUDIO_FEEDBACK

Each key press generates an authentic 650 Hz sine-wave tone — the classic frequency used in vintage CW radio operation. Toggle audio with the speaker icon in the nav bar.

🔤

SUPPORTED_CHARS

Full International Morse Code — A–Z, 0–9, and common punctuation marks including period, comma, question mark, slash, parentheses, and more.
